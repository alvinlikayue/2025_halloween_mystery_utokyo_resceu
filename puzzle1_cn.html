🧩 謎題1：命運的交叉點

紅色的線拼出「UTOKYO」。
藍色的線拼出「TOYAMA」。
那麼，綠色的線拼出的是什麼？

🧩 解題思路

這張表是一個字母方格。每一條線依序連結字母，組成一個單字。

  • 🔴 紅線：U → T → O → K → Y → O → UTOKYO
  • 🔵 藍線：T → O → Y → A → M → A → TOYAMA
  • 🟢 綠線：K → A → G → R → A → KAGRA

柯南：「K → A → G → R → A。綠線拼出的，是『KAGRA』！」

🔭 延伸閱讀：什麼是 KAGRA？

KAGRA 是位於日本岐阜縣神岡山區地下的大型觀測設施。
這是一個致力於 觀測宇宙 的研究計畫—— 透過數公里長的雷射通道， 偵測來自宇宙深處的 微小振動。

KAGRA 與世界各地的研究者合作， 持續挑戰觀測那 「宇宙的聲音」—— 那是人類從未聽過的低語。
