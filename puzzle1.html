🧩 謎そのいち：運命の交差点

赤い線のときは「東大」、紺色線のときは「富山」。
では、緑の線のときは何？

🧩 謎の構造

この表は五十音表（あいうえお表）。縦は「行（あ・か・さ・た・な…）」、横は「段（a・i・u・e・o）」を表している。

線は、それぞれ音の位置をたどって、日本語の言葉を描いていた。

  • 🔴 赤い線：「と → う → だ → い」＝東大
  • 🔵 紺色線：「と → や → ま」＝富山
  • 🟢 緑色線：「か → ぐ → ら」＝KAGRA

コナン：「……“か → ぐ → ら”。 緑の線は『KAGRA（かぐら）』だ！」

🔭 もっと知りたい！KAGRAとは？

KAGRA（カグラ）は、岐阜県・神岡の山の地下に広がる、 光で宇宙を感じ取ろうとする研究施設です。
地面の下に長いトンネルをつくり、そこに光を通して 宇宙から届く“わずかなゆらぎ”を観測しています。

世界中の研究者たちと協力し、 まだ誰も見たことのない「宇宙の音」を聴こうとする―― それが KAGRA の挑戦です。
