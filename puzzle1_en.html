🧩 Puzzle 1: Crossroads of Fate

The red line makes “UTOKYO.”
The blue line makes “TOYAMA.”
Then, what word does the green line make?

🧩 How It Works

This grid is an alphabet chart. Each line traces letters in order to form a word.

  • 🔴 Red line: U → T → O → K → Y → O → UTOKYO
  • 🔵 Blue line: T → O → Y → A → M → A → TOYAMA
  • 🟢 Green line: K → A → G → R → A → KAGRA

Conan: “...K → A → G → R → A. The green line spells ‘KAGRA’!”
